Loaders and Savers¶

Loaders and savers reduce the amount of boilerplate needed load and save resources in meillionen by providing loaders and savers for commonly used numerical python packages such as xarray, pandas and netcdf

class meillionen.io.LandLabLoader¶

A loader to load raster files into landlab grid objects

static load(dem_resource)¶

Loads a raster file into a landlab grid

Parameters

dem_resource – metadata describing where the raster is located and its format

Returns

A landlab grid

class meillionen.io.LandLabSaver¶

A saver to persist landlab grid objects

static save(dem_resource, mg, variable: str)¶

Saves a model grid to a location and in a format specified by a resource

Parameters

  • dem_resource – metadata describing the location and format to save the data in

  • mg – the model grid object

  • variable – name of variable in model grid to save

class meillionen.io.NetCDF4Saver(sink, sink_schema)¶

A saver that allows incrementally saving array slices to a NetCDF file

Intended to allow the saving or large arrays that may not fit into memory

set_slice(array: xarray.core.dataarray.DataArray, **slices: Dict[str, Union[int, slice]])¶

Save an array to a slice of a NetCDF variable

Parameters

  • array – A dimensioned array

  • slices – Where to save the dimensioned array to. Dimensions of array passed in and NetCDF variable names must match

class meillionen.io.PandasLoader¶

Loads a resource into pandas

Parameters

class meillionen.io.PandasSaver¶

A saver to persist a pandas dataframe to the location and format specified by a resource

Parameters

  • sink_resource – metadata describing the location and format to save a dataset to

  • data – the data to be saved
